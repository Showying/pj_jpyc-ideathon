# デモ操作マニュアル (Demo Script)

## 概要
本モックアップを使用したプレゼンテーション用の操作ガイドです。
画面上の赤丸番号（①〜⑦）に従って操作することで、スムーズにストーリーを伝えることができます。

## 準備
1.  `frontend/index.html` をブラウザで開く。
2.  画面サイズを「スマホサイズ」または「狭めのウィンドウ」にすると最適に表示されます。
3.  スーパーリロードを行い、最新の画像・キャッシュ状態にします。

## 操作フロー (Start to Finish)

| Step | アクション (操作) | 画面・状態 | トーク/ナレーションのポイント |
| :--- | :--- | :--- | :--- |
| **①** | **Manageタブを見る** | アノテーション①が表示中 | 「まず、クリエイターが抱える『業務の煩雑さ(Operational Pain)』を解決します。」 |
| **②** | **`Generate Idea` をクリック** | AIプロンプトが走り、ログが表示される | 「AI店長にお願いするだけ。裏側では一瞬で工場や配送業者を検索し、提携(Link)を結びます。」 |
| **③** | **コスト内訳を見る** | `Sci-Fi Bonsai Art Book` が表示 | 「商品ができました。同時に、製造原価800円もクリアになっています。」 |
| **④** | **手数料を見る(赤字)** | 取り消し線のPF手数料 | 「ここを見てください。プラットフォーム手数料はゼロ。振込手数料もゼロです。」 |
| **⑤** | **利益を見る** | `Your Profit +500` | 「だから、クリエイターの利益が最大化されます。しかも即時着金です。」 |
| **⑥** | **`Publish` をクリック** | **Marketタブへ自動遷移** | 「ワンクリックで世界中に公開。契約書(Smart Contract)も同時にデプロイされました。」 |
| **⑦** | **商品を買う** | Market画面。Art Bookをクリック | 「買い手が現れました。購入ボタンを押すと…（コイン分裂アニメーション）」 |
| **⑧** | **ポップアップ「OK」** | 「購入完了」ポップアップが出現 | 「お金が物理的に分割されて飛んでいきましたね。ここで管理画面に戻ってみましょう。」 |
| **⑨** | **ログ確認 (自動)** | **Manageタブへ自動遷移しログが流れる** | 「戻った瞬間を見てください。スマートコントラクトが『入金確認→工場へ送金→配送手配』を全自動で完了させました。私が寝ていても、ビジネスは止まりません。」 |

## 終了後
- リロードすれば初期状態に戻り、何度でもデモ可能です。
- 画像は `Sci-Fi Bonsai` シリーズで統一されています。
